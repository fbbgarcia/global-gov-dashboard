# Global Government Dashboard

Explore global histories and trends of democratic and dictatorial regimes with our [dashboard](https://fbbgarcia-global-government-dashboard.share.connect.posit.cloud/)!

Visualizations, summaries, and analysis are based on the [TidyTuesday Democracy and Dictatorship](https://github.com/rfordatascience/tidytuesday/tree/main/data/2024/2024-11-05) and [Powell and Thyne Coup d’État](https://www.uky.edu/~clthyn2/coup_data/home.htm) datasets.
